# PathAble-TS

Monorepo for public domain TypeScript npm packages in the Effect ecosystem. Each `packages/*` directory is a publishable library.

## Tooling Setup

Tools are set up one at a time, following the list in
`scratchpad/tools.txt`. Create a commit after each tool.

Completed: corepack, pnpm, typescript, fallow, eslint, renovate, husky, tsgo.
